# Employee Management System

A backend project built with **Java 21, Spring Boot, JPA, MySQL, and Flyway**.  
It provides REST APIs to manage employees and departments, with database migrations and validation.

---

## 🚀 Features
- CRUD operations for employees and departments
- Database migrations with Flyway
- Validation with Hibernate Validator
- RESTful endpoints tested via Postman
